## 3.2 프로필 관리

---

### FR-P2-005 내 프로필 조회

**설명:** 로그인한 사용자는 자신의 프로필 정보를 조회할 수 있어야 한다.

**기능 상세:**
- 사용자(STUDENT/INSTRUCTOR/ADMIN)는 자신의 닉네임, 이메일, 역할, 가입일을 확인할 수 있다.
- 강사는 추가로 자기소개(bio)를 확인할 수 있다.

**제약 조건:**
- 로그인한 사용자만 조회 가능하다.

**예외 처리:**
- 비인증 요청 시: `"로그인이 필요합니다"` (401 Unauthorized)

---

### FR-P2-006 프로필 수정

**설명:** 로그인한 사용자는 자신의 닉네임 및 자기소개를 수정할 수 있어야 한다.

**기능 상세:**
- 사용자(STUDENT/INSTRUCTOR)는 닉네임을 수정할 수 있다.
- 강사(INSTRUCTOR)는 자기소개(bio, 최대 500자)를 추가로 수정할 수 있다.

**제약 조건:**
- 닉네임: 최소 2자, 최대 20자, 중복 불가, 특수문자 불가

**예외 처리:**
- 이미 사용 중인 닉네임 입력 시: `"이미 사용 중인 닉네임입니다"` (409 Conflict)

---

### FR-P2-007 비밀번호 변경

**설명:** 로그인한 사용자는 자신의 비밀번호를 변경할 수 있어야 한다.

**기능 상세:**
- 사용자는 현재 비밀번호와 새 비밀번호를 입력하여 변경을 요청한다.
- 시스템은 현재 비밀번호를 검증한 후 새 비밀번호로 업데이트한다.
- 비밀번호 변경 시 해당 사용자의 모든 Refresh Token을 무효화한다.

**제약 조건:**
- 새 비밀번호: 최소 8자, 영문 + 숫자 조합 필수, 최대 64자
- 현재 비밀번호와 동일한 비밀번호로 변경 불가

**예외 처리:**
- 현재 비밀번호 불일치 시: `"현재 비밀번호가 올바르지 않습니다"` (400 Bad Request)
- 새 비밀번호 형식 오류 시: `"비밀번호는 영문과 숫자를 포함하여 8자 이상이어야 합니다"` (400 Bad Request)
- 현재 비밀번호와 동일한 비밀번호 입력 시: `"현재 비밀번호와 동일한 비밀번호로 변경할 수 없습니다"` (400 Bad Request)

### FR-P2-008 회원 탈퇴

**설명:** 로그인한 사용자는 자신의 계정을 탈퇴할 수 있어야 한다.

**기능 상세:**
- 사용자는 비밀번호를 입력하여 탈퇴를 요청한다.
- 시스템은 탈퇴 확인 후 해당 계정을 비활성화(소프트 삭제) 처리한다.
- 탈퇴 시 해당 사용자의 모든 Refresh Token을 무효화한다.
- 탈퇴한 사용자의 게시된 질문·답변은 '탈퇴한 사용자'로 표시되며 삭제되지 않는다.
- 강사가 탈퇴하는 경우, 해당 강사의 APPROVED 상태 강의는 HIDDEN 처리된다.

**제약 조건:**
- 로그인한 사용자만 탈퇴 가능하다.

**예외 처리:**
- 비밀번호 불일치 시: `"비밀번호가 올바르지 않습니다"` (400 Bad Request)
