# LoopLearn — front/CLAUDE.md

## 기술 스택
- **Framework:** Flutter (단일 코드베이스)
- **지원 플랫폼:** Web(PC 브라우저, Chrome 권장) + 태블릿 앱(Android/iOS)
- **태블릿 앱:** 가로 모드(Landscape) **전용** — `SystemChrome.setPreferredOrientations`로 강제 고정
- **API 통신:** REST / JSON (`Base URL: /api`)

## 페이지 구성

| 페이지 | 경로(예시) | 설명 |
|--------|-----------|------|
| 강의 목록 | `/` | 강의 카드 목록, 검색(이름·강사), 태그 필터 |
| 강의 상세 | `/lectures/:id` | 강의 정보, 파트 목록, 수강 신청 버튼 |
| 강의 시청 | `/lectures/:id/parts/:partId` | 영상 + 오른쪽 패널(강의 목록 or 도안) |
| 내 강의 | `/my-lectures` | 수강 중 강의 목록, 진도율 |
| Q&A 목록 | `/lectures/:id/questions` | 질문 목록 |
| Q&A 상세 | `/questions/:id` | 질문 상세 |
| Q&A 작성 | `/lectures/:id/questions/new` | 질문 작성 폼 |

## 강의 시청 화면 레이아웃
```
┌──────────────────────────────┬──────────────────┐
│                              │  [강의 목록 탭]  │
│     YouTube 영상 재생        │  [도안 보기 탭]  │
│                              │                  │
│                              │  (선택된 패널    │
│                              │   내용 표시)     │
└──────────────────────────────┴──────────────────┘
```
- 오른쪽 패널: **강의 목록 보기** ↔ **도안 보기** 탭 전환
- 도안이 없는 강의는 도안 탭 미표시

## 도안 보기 — 두 가지 모드

### 강의 단위 도안 (PATTERN 유형)
- `GET /api/lectures/{lectureId}/patterns` 호출
- 영상 없이 전체 도안 텍스트 표시

### 파트 단위 도안 (STITCH_BASICS / PROJECT_CLASS)
- `GET /api/parts/{partId}/patterns` 호출
- YouTube 영상 현재 재생 시간을 polling하여 `startTime ≤ currentTime < endTime`인 행 하이라이트

## 수강 정보 — 로컬 스토리지 (P1)
P1에서는 서버 수강 API를 사용하지 않고 **브라우저 로컬 스토리지**에 저장.

저장 구조 예시:
```json
{
  "enrollments": [
    {
      "lectureId": 1,
      "completedParts": [2, 3],
      "totalParts": 5,
      "progress": 40
    }
  ]
}
```

- 새로고침·브라우저 종료 후에도 유지
- 수강 취소 시 해당 lectureId 항목 + 진도 데이터 함께 삭제
- P2에서 `/api/enrollments` 서버 API로 전환 예정

## 단수 카운터
- 도안 화면 하단에 표시
- `[−]` / `[+]` 버튼으로 1 감소·증가, 최솟값 0
- 초기값: 0, 값은 로컬에 저장 (강의·파트 단위 키 권장)

## 도안 기호 설명
- 도안 텍스트에서 기호 탭/클릭 → `GET /api/pattern-symbols/{symbol}` 호출 → 팝업 표시
- 예: `sc` 클릭 → "짧은뜨기" 팝업

## Q&A 작성 제약
- 제목: 필수, 최대 100자
- 내용: 필수, 최대 1000자
- 이미지: 선택, 1장, 최대 5MB, jpg/png/gif
- P1: 익명 작성 (userId 미전송)

## 플랫폼별 유의사항
- **웹:** 화면 방향 제한 없음, `localStorage` 사용 가능
- **앱:** 가로 모드 전용 강제 고정, `shared_preferences` 등으로 로컬 저장
- 단일 코드베이스이므로 플랫폼 분기는 `kIsWeb` 또는 `Platform` 체크 사용

## 비기능 요구사항
- 강의 목록·검색·상세: 응답 후 **2초 이내** 렌더링
- 도안 탭 전환: **즉각적** (별도 API 호출 없이 캐시 활용 권장)
- 태블릿 가로 모드 레이아웃에 최적화
